# Security Policy

## Supported Versions

Please use the latest version of GlyphOS to ensure you have the most recent security updates.

| Version | Supported |
| ------- | ------------------ |
| 1.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in GlyphOS, please report it responsibly.

**Do not open a public issue.**

Instead, please email us directly with the details of the vulnerability. We will review it and get back to you as soon as possible.

## Security Best Practices for Users

*   **API Keys**: GlyphOS is designed to store API keys locally in your browser. Never share your keys or commit them to public repositories.
*   **Local Deployment**: When running GlyphOS locally, ensure your environment is secure.

Thank you for helping keep GlyphOS secure!
