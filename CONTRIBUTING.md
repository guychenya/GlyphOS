# Contributing to GlyphOS

Thank you for your interest in contributing to **GlyphOS**! We welcome contributions from everyone.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/your-username/GlyphOS.git
    cd GlyphOS
    ```
3.  **Create a new branch** for your feature or bug fix:
    ```bash
    git checkout -b feature/my-new-feature
    ```

## Making Changes

*   **Code Style**: Please maintain consistency with the existing code style.
*   **Commits**: Write clear and concise commit messages.

## Submitting a Pull Request

1.  Push your changes to your fork:
    ```bash
    git push origin feature/my-new-feature
    ```
2.  Open a **Pull Request** on the main GlyphOS repository.
3.  Provide a clear description of your changes and why they are necessary.

## Reporting Issues

If you encounter any bugs or have feature requests, please open an issue on the [GitHub Issues](https://github.com/guychenya/GlyphOS/issues) page.
